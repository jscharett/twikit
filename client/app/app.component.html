Full-stack Web Developer Exercise

The goal of this exercise is to asses your code quality. Most important factors are structure, readability and efficiency.

Requirements

Create a simple nodejs based backend api that can accept a CSV file upload, parse it and return a structured JSON of the data. Create a html-page that consumes that api by allowing the user to select a CSV file that will be uploaded and dynamically displays the returned data in a tabular format.

The project should also contain a README.md which details all the steps to run this example

Scope

You are allowed to use any publicly available package or library. But please provide details for installation in the README. Try to limit the requirement for globally installed packages.

We will test this with a variety of csv files, these files will include edge case (quotes, escaping, encoding, etc...)

Delivery

Before beginnen please send us a estimate of how long (hours) you think you will need to complete it. And when you will be able to deliver the project.

The end result is a git project we can access or a zip file containing all the necessary files.
